# what to do after every change.

- git commit and push
- make the xpi

# version numbering system

The version has three level:
- major: this shall be increased only if by user request
- minor: increase this number after considerable changes
- patch: increase this number for all other less important changes

Prepend the version number to all commit descriptions.
Also add git tags with version numbers.
Put the version number in the xpi filename.
